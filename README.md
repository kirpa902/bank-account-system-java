# Bank Account System (Java)

## Description
This Java application simulates a simple banking system that allows users to manage bank accounts.  
It supports basic operations such as depositing money, withdrawing money with validation, and transferring funds between accounts. The program also displays account details.

## Features
- Deposit money with validation  
- Withdraw money with balance checks  
- Transfer funds between accounts  
- Print account statements  

## Concepts Used
- Object-Oriented Programming (OOP)  
- Encapsulation  
- Conditional logic
